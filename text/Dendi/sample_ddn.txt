( MAT 1:1 ) Yesu Mɛsiyɑ ko ɑ̀ ci Dɑfidi ize, Abulɛmɑ ize, ɑ̀ cɛɛ kɑ cɛɛ yom tirɑ neyɑ.
( MAT 1:2 ) Abulɛmɑ nɑ Isɑɑkɑ hɛi. Isɑɑkɑ nɑ Yɑkɔfu hɛi. Yɑkɔfu mo nɑ Yudɑ ndɑ ɑ̀ bɛɛrɛ yom ndɑ ɑ̀ cɛɛnɛ yom hɛi.
( MAT 1:3 ) Yudɑ mo nɑ Fɑresi ndɑ Zerɑ hɛi Tɑmɑrɑ gɑɑ. Fɑresi nɑ Esirɑm hɛi. Esirɑm nɑ Arɑm hɛi.
( MAT 1:4 ) Arɑm mo nɑ Aminɑdɑbu hɛi. Aminɑdɑbu mo nɑ Nɑɑsɑm hɛi. Nɑɑsɑm nɑ Sulumi hɛi.
( MAT 1:5 ) Sulumi mo nɑ Boɑzu hɛi Lɑhɑbu gɑɑ. Boɑzu nɑ Obe hɛi Lutɑ gɑɑ. Obe mo nɑ Yɑsɑi hɛi.
( MAT 1:6 ) Yɑsɑi nɑ Kpe Dɑfidi hɛi. Kpe Dɑfidi mo nɑ Sɑlomɔ hɛi Uri wɛndɛ zeenɑ gɑɑ.
( MAT 1:7 ) Sɑlomɔ nɑ Lobuɑm hɛi. Lobuɑm nɑ Abiɑ hɛi. Abiɑ nɑ Asɑ hɛi.
( MAT 1:8 ) Asɑ mo nɑ Yosufɑ hɛi. Yosufɑ nɑ Yorɑm hɛi. Yorɑm nɑ Uziɑ hɛi.
( MAT 1:9 ) Uziɑ mo nɑ Yotɑm hɛi. Yotɑm nɑ Ahɑzɑ hɛi. Ahɑzɑ nɑ Ezeciɑ hɛi.
( MAT 1:10 ) Ezeciɑ mo nɑ Mɑnɑse hɛi. Mɑnɑse nɑ Amɔ hɛi. Amɔ nɑ Yosiɑ hɛi.
( MAT 1:11 ) Yosiɑ mo nɑ Yekoniɑ ndɑ ɑ̀ cɛɛnɛ yom hɛi sɑɑ kɑ ǹ nɑ ǹ dii kɑ kpeindei Bɑbiloni lɑɑbu.
( MAT 1:12 ) Ǹ diiyom kɑ kpeindei Bɑbiloni lɑɑbu bɑndɑ no, Yekoniɑ mo nɑ Sɑlɑtilu hɛi. Sɑlɑtilu mo nɑ Zorobɑbe hɛi.
( MAT 1:13 ) Zorobɑbe nɑ Abiude hɛi. Abiude nɑ Eliɑkim hɛi. Eliɑkim mo nɑ Azoro hɛi.
( MAT 1:14 ) Azoro mo nɑ Sɑdoki hɛi. Sɑdoki nɑ Akim hɛi. Akim nɑ Heludɑ hɛi.
( MAT 1:15 ) Heludɑ nɑ Heliɑzuru hɛi. Heliɑzuru mo nɑ Mɑtɑm hɛi. Mɑtɑm nɑ Yɑkɔfu hɛi.
( MAT 1:16 ) Yɑkɔfu mo nɑ Yosɛfu hɛi, Mɑriɑmɑ kunye no. Mɑriɑmɑ nɑ Yesu hɛi, ngɑ kɑ ǹ nɑ ɑ̀ cɛɛ Mɛsiyɑ.
( MAT 1:17 ) Cɛɛ kɑ cɛɛ yom di kulu zɑ Abulɛmɑ gɑɑ kɑlɑ Dɑfidi gɑɑ, ǹ kulu cɛɛ wei cindi tɑɑci no. Zɑ Dɑfidi mo gɑɑ kɑlɑ sɑɑ kɑ ǹ nɑ ǹ dii kɑ kpeindei Bɑbiloni lɑɑbu di, cɛɛ wei cindi tɑɑci no. Zɑ ǹ kpeindei Bɑbiloni lɑɑbu di mo kɑlɑ Mɛsiyɑ gɑɑ cɛɛ wei cindi tɑɑci no.
( MAT 1:18 ) Yesu Mɛsiyɑ hɛiyom zɑngɑ ɑ̀ tɛ ndɑ neyɑ. À nyɑ Mɑriɑmɑ yɑ Yosɛfu hundiyo no. Hɑli ǹ meigu cɔfɔ Biyɑ Hɑlɑɑlɑ nɑ gunde dɑm Mɑriɑmɑ gɑɑ.
( MAT 1:19 ) Ammɑ ɑ̀ sɔgɔ Yosɛfu ci bɔrɔ cɛɑntɛ susu. À si bɑɑ ngɑ mɑ ɑ̀ hɑwɑndi. À go miilɑ ngɑ mɑ ɑ̀ fee tugu kunɑ.
( MAT 1:20 ) Kɑ ɑ̀ go hɛ nge yom di lɑsɑbu tɛ, cɔfɔ Kpe ŋmɔne dɔntɔneize fɔ hunutɛrɛ ɑ̀ sɛ hɑndiri kunɑ. À cii: Yosɛfu, Dɑfidi ize, si hɑmburu ndɑ n hundiyo Mɑriɑmɑ zɑɑyom ɑ̀ mɑ ci n wɛndɛ. N mɑ si hɑmburu zɑmɑ hɛ kɑ ǹ dɑm ɑ̀ gɑɑ, Biyɑ Hɑlɑɑlɑ yɑ nɑ ɑ̀ dɑm.
( MAT 1:21 ) À kɑɑ koo hɑibɔrɔ fɔ hɛi. Ǹ gɑ ɑ̀ mɑɑ no Yesu zɑmɑ ngɑ yɑ gɑ ngɑ jɑmɑ fɑɑbɑ ndɑ ǹ dulum yom.
( MAT 1:22 ) Hɛ wɔ kulu tɛ mo no zɑmɑ hɛ kɑ Kpe cii ɑndebi gɑɑ mɑ tɔ sɛ.
( MAT 1:23 ) Andebi di cii: “Gunɑ, hundiyo kɑ ɑ̀ si hɑru bei, ɑ̀ kɑɑ kunu. À bine kɑɑ koo hɑibɔrɔ fɔ hɛi. Ǹ gɑ ɑ̀ cɛɛ Emɑnuɛli.” À mɑɑ feeriji yɑ ci Ikpɛ goono i bɑndɑ.
( MAT 1:24 ) Kɑ Yosɛfu tunu ngɑ jiibi kunɑ, ɑ̀ tɛ zɑngɑ Ikpɛ dɔntɔneize di nɑ ɑ̀ no fɔndɔ kɑ ɑ̀ mɑ ngɑ hundiyɑ hiji.
( MAT 1:25 ) Ammɑ Yosɛfu si ɑ̀ bei kɑlɑ ɑ̀ nɑ koo sintine hɛi. À nɑ ɑ̀ mɑɑ dɑm Yesu.
( MAT 2:1 ) Ǹ nɑ Yesu hɛi Bɛtɛlɛm kpɑɑrɑ Yude lɑɑbu gɑndɑkpɛ Hirodu kpetɛrɛ wɑtɛ. Cɔfɔ beirekpɛ yom hunu zɑ weenuhune kɑ kɑɑ Yerusɑlɛm wɑngɑrɑ.
( MAT 2:2 ) Ǹ cii: Mɑni Yuifu yom gɑndɑkpɛ di kɑ ǹ hɛi goono? I di ɑ̀ hɑndɑriyɑize weenuhune hɛrɛ. I kɑɑ kɑ ɑ̀ bɛɛrɑndi no.
( MAT 2:3 ) Ammɑ kɑ Kpe Hirodu mɑɑ ngɑ di, bɑɑru di nɑ ɑ̀ butugu, ngɑ ndɑ Yerusɑlɛm kulu.
( MAT 2:4 ) Cɔfɔ ɑ̀ nɑ gɑrɑndikɔ jinɑ yom meigu ndɑ Moisi fɔndɑ cooɑndikɔ yom kɑ ǹ hã nungu kɑ ɑ̀ himɑ ǹ mɑ Mɛsiyɑ hɛi.
( MAT 2:5 ) Ǹ cii ɑ̀ sɛ: Bɛtɛlɛm kpɑɑrɑ Yude lɑɑbu zɑmɑ ɑndebi nɑ wɔne hɑntum kɑ cii,
( MAT 2:6 ) Ni Bɛtɛlɛm, Yude lɑɑbu n mɑnɑ ci Yude kpɑɑrɑ beeri yom ŋmɔne icɛnɑ. N kunɑ no meirekpɛ fɔ kɑɑ hunu. À gɑ ɑ̀ ŋmɔne Isirɑilɑ jɑmɑ kurum.
( MAT 2:7 ) Hirodu nɑ beirekpɛ yom cɛɛ tugu kunɑ kɑ ǹ hã cimi cimi lɔkɑci kɑ hɑndɑriyɑize di hunutɛrɛ.
( MAT 2:8 ) À nɑ ǹ dɔntɔm Bɛtɛlɛm kpɑɑrɑ kɑ cii: Wɑ kpei hã ndɑ lɑɑkɑli kɑ simbɑ ndɑ koo di. De wom di ɑ̀, wɑ nɑm yɑ mɑɑ. Hɛɛ mo yɑ kpei kɑ ɑ̀ bɛɛrɑndi.
( MAT 2:9 ) Ngei mo kɑ ǹ mɑɑ kpe di sɛ, ǹ kpei. Cɔfɔ hɑndɑriyɑizɑ kɑ ǹ di weenuhune hɛrɛ, ɑ̀ go ǹ jine bɛɛnɛ kɑlɑ ɑ̀ kɑɑ cɛ nungu kɑ koo di goono.
( MAT 2:10 ) Sɑɑ kɑ ǹ di hɑndɑriyɑizɑ cɛ, ǹ fɑihã ndɑ binekɑɑne bɑmbɑtɑ.
( MAT 2:11 ) Kɑ ǹ huro fuuɑ kunɑ, ǹ di koo di ndɑ ɑ̀ nyɑ Mɑriɑmɑ. Ǹ jɛndɛbiri kɑ ɑ̀ bɛɛrɑndi. À bɑndɑ ǹ nɑ ngei hɑibɑrikɑ feeri. Ǹ nɑ ɑ̀ no wurɑ ndɑ dugubi ndɑ wɑidi kɑɑnɔ kɑ ǹ cii ɑ̀ sɛ mire.
( MAT 2:12 ) Hɑndiri kunɑ Ikpɛ nɑ beirekpɛ yom kɑɑbɑɑru ǹ mɑ si ye Hirodu dɔ. Ǹ ye ngei lɑɑbu ndɑ fɔndɔ ŋmɑni fɔ.
( MAT 2:13 ) Sɑɑ kɑ beirekpɛ yom di dirɑ, cɔfɔ Kpe ŋmɔne dɔntɔneize fɔ hunutɛrɛ Yosɛfu sɛ hɑndiri kunɑ. À cii: Tunu kɑ koo cɛnɑ di ndɑ ɑ̀ nyɑ zɑɑ kɑ zuuru ndei kɑ kpei Eziputu lɑɑbu. N mɑ gɔrɔ Eziputu lɑɑbu kɑlɑ yɑ cii n sɛ zɑmɑ Hirodu bɑ koo cɛnɑ di ceeci kɑ ɑ̀ wi.
( MAT 2:14 ) Yosɛfu mo tunu kɑ koo di ndɑ ɑ̀ nyɑ zɑɑ ciini kɑ kpei ndei Eziputu lɑɑbu.
( MAT 2:15 ) Nungu di no ɑ̀ goono kɑlɑ Hirodu bu zɑmɑ hɛ kɑ Kpe cii ɑndebi gɑɑ mɑ tɔ sɛ. À cii: “A nɑ ɑ koo cɛɛ ɑ̀ mɑ hunu Eziputu lɑɑbu”.
( MAT 2:16 ) Kɑ Hirodu bei beirekpɛ yom di nɑ ngɑ zɑmbɑ, ɑ̀ bine tɔ hɑli ǹ ci mɔsɔ. Cɔfɔ ɑ̀ nɑ beerem dɑm ǹ mɑ koo hɑibɔrɔ yom kulu kɑ ǹ goono Bɛtɛlɛm kpɑɑrɑ ndɑ ɑ̀ hundɑntɛ kulu wi. Ǹ mɑ koo hɑibɔrɔ yom wi zɑ jiiri hinkɑ ize mee kɑ ye gɑndɑ ndɑ zɑngɑ ɑ̀ lɑsɑbu beirekpɛ yom di sendɑ bɔm.
( MAT 2:17 ) Bine, hɛ kɑ ɑndebi Yeremi cii mɑ tɔ, kɑ ɑ̀ cii:
( MAT 2:18 ) “Ǹ mɑɑ jinde fɔ Lɑmɑ kpɑɑrɑ, bɛrɛɛ ndɑ hɛ̃ni ndɑ zɑlibɑ yom bɑmbɑtɑ. Lɑsɛli go hɛ̃ ngɑ koo yom sɛ. À mɑnɑ du suurɑndiyom zɑmɑ ɑ̀ koo yom bu.”
( MAT 2:19 ) Ammɑ kɑ Hirodu bu, cɔfɔ Kpe ŋmɔne dɔntɔneize fɔ hunutɛrɛ Yosɛfu sɛ hɑndiri kunɑ Eziputu lɑɑbu.
( MAT 2:20 ) À cii: Tunu kɑ koo di ndɑ ɑ̀ nyɑ zɑɑ kɑ kpeindei Isirɑilɑ lɑɑbu zɑmɑ beerem kɑ ǹ go koo cɛnɑ di hundi ceeci, ǹ bu.
( MAT 2:21 ) Yosɛfu bine tunu kɑ koo cɛnɑ di ndɑ ɑ̀ nyɑ zɑɑ kɑ kɑɑ Isirɑilɑ lɑɑbu.
( MAT 2:22 ) Ammɑ kɑ Yosɛfu mɑɑ ǹ cii Akelu go kpetɛrɛ ŋmɑɑ Yude lɑɑbu ɑ̀ bɑɑbɑ Hirodu guribi, ɑ̀ hɑmburu ngɑ mɑ kpei nungu di. Kɑ Ikpɛ nɑ ɑ̀ kɑɑbɑɑru hɑndiri kunɑ, ɑ̀ nɑ ngɑ bɔm cendi kɑ kpei Gɑlile lɑɑbu.
( MAT 2:23 ) À kɑɑ gɔrɔ kpɑɑrɑ fɔ kɑ ǹ cii ɑ̀ sɛ Nɑzɑrɛtu. À kɑɑ gɔrɔ Nɑzɑrɛtu zɑmɑ hɛ kɑ ǹ cii ɑndebi yom mee gɑɑ di mɑ tɔ sɛ kɑ ǹ cii: "Ǹ gɑ ɑ̀ cɛɛ Nɑzɑrɛtu bɔrɔ”.
( MAT 3:1 ) Hɑnu nge yom di gɑɑ, Mirɑndikɔ Yohɑnnɑ kɑɑ ko wɑɑzu Yude lɑɑbu gɑnji kunɑ.
( MAT 3:2 ) À cii: Wɑ tuubi zɑmɑ bɛɛnɛ kpetɛrɛ mɑɑni.
( MAT 3:3 ) Yohɑnnɑ yɑ ci ngɑ kɑ ɑndebi Esɑyi nɑ ɑ̀ sendi tɛ kɑ cii: Bɔrɔ fɔ go kuwɑ gɑnji kunɑ. À jinde cii: Wom mɑ Kpe fɔndɔ sɔlu. Wom mɑ ɑ̀ fɔndɔize yom cɛɑndi.
( MAT 3:4 ) Yohɑnnɑ bumbum go yoo hɑmini kpɛ dɑm ndɑ kunɑcendi kuuru ŋmɔne ngɑ cɛntɛ gɑɑ. À ŋmɑɑri yɑ ci doo ndɑ gɑnji yu.
( MAT 3:5 ) Bine, Yerusɑlɛm wɑngɑrɑ bɔrɔ yom ndɑ Yude lɑɑbu kulu bɔrɔ yom ndɑ lɑɑbu yom kulu kɑ ǹ go mɑɑni Yudum isɑ gɑɑ, ǹ kulu hunu kɑ kpei ɑ̀ dɔ.
( MAT 3:6 ) À nɑ ǹ mirɑndi Yudum isɑ kunɑ kɑ ǹ nɑ ngei dulum yom cii.
( MAT 3:7 ) Ammɑ ɑ̀ di Fɑrisi fɔndɔ bɔrɔ boobo ndɑ Sɑdusi fɔndɔ bɔrɔ yom go kɑɑ ngɑ mirɑndiyomɑ dɔ. À cii ǹ sɛ: Gɑzɑmɑ dimi yom, mee yɑ nɑ wom kɑɑbɑɑru wom mɑ zuuru binetɔnɛ kɑ ɑ̀ go kɑɑ sɛ?
( MAT 3:8 ) Ngɑ di sɛ wɑ wom tɛgbei cɛbɛ kɑ ɑ̀ sɑbɑ ndɑ tuubiyom.
( MAT 3:9 ) Wom bumbum mɑ si lɑsɑbu kɑ cii: I gundɑ i bɑɑbɑ Abulɛmɑ. A go cii wom sɛ kɑ Ikpɛ hini kɑ tondi yom wɔ bɛrɛ kɑ tɛ ize yom Abulɛmɑ sɛ.
( MAT 3:10 ) Ǹ bine nɑ deesi dɑm tuurinyɑ yom kɑɑji gɑɑ kɔ. Ngɑ di sɛ tuurinyɑ kulu kɑ ɑ̀ si ize hinno hɛi, ǹ go kɑ ɑ̀ dumbu no kɑ ɑ̀ tɔnu.
( MAT 3:11 ) Hɛɛ wɔ, ɑ go wom mirɑndi ndɑ hɑri tuubiyom sɑbu sɛ. Ammɑ bɔrɔ kɑ ɑ̀ go kɑɑ ɑ bɑndɑ ɑ̀ bisɑ hɛɛ gɑɑbi. Yɑ nɑ tɔ yɑ ɑ̀ tɑɑmu yom zɑɑ. Ngɑ wɔ, ɑ̀ kɑɑ wom mirɑndi no ndɑ Biyɑ Hɑlɑɑlɑ ndɑ nini mo.
( MAT 3:12 ) À funsu gɑɑsu goono ɑ̀ kɑmbɛ. À gɑ ngɑ fɑruyom dɔ hinnɑndi sã sã kɑ ngɑ mɔsɔngu dɑm boo. Ammɑ duuɑ wɔ, ɑ̀ gɑ ɑ̀ tɔnu ndɑ nini kɑ ɑ̀ si wi.
( MAT 3:13 ) Sɑɑ di no Yesu hunu zɑ Gɑlile kɑ kɑɑ Yudum isɑ Yohɑnnɑ dɔ, ɑ̀ mɑ ngɑ mirɑndi.
( MAT 3:14 ) Ammɑ Yohɑnnɑ go hungu ɑ̀ sɛ ndɑ gɑɑbi kɑ cii: Hɛɛ wɔ ɑ̀ himɑ ni, n mɑ ɑ mirɑndi, cɔfɔ n kɑɑ ɑ dɔ?
( MAT 3:15 ) Yesu tu ɑ̀ sɛ kɑ cii: Nɑm ɑ̀ mɑ tɛ mɑɑsɑ zɑmɑ yɑ no ǹ nɑ ɑ̀ sɑsɑbɑndi i sɛ kɑ i mɑ cɛyom susu kulu tɔnɑndi. Yohɑnnɑ mɑnɑ hungu ɑ̀ sɛ kpɛɛnɛ.
( MAT 3:16 ) Kɑ Yohɑnnɑ nɑ Yesu mirɑndi, ɑ̀ hunu mɑɑsɑ mɑɑsɑ hɑri di kunɑ. Cɔfɔ bɛɛnɛ feeri ɑ̀ sɛ. À di Ikpɛ Biyɑ go zumbu kɑ sɔm ngɑ bɔm zɑngɑ gutu.
( MAT 3:17 ) Cɔfɔ jinde fɔ mo hunu zɑ bɛɛnɛ kɑ cii: Wɔne yɑ ci ɑ Ize Binegɑnji kɑ ɑ̀ gɑɑ ɑ du binekɑɑne.
